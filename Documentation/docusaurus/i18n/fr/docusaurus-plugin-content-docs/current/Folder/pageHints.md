# How to insert hints?

You can add tips, info, hints, warnings, using this syntax:  

```
:::note A note
A note
:::
```

:::note A note
A note
:::

:::info An info
An info
:::

:::warning A warning
A warning
:::

:::caution A caution
A caution
:::

:::tip A tip
A tip
:::
